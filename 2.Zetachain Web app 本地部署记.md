解决系统内 yarn 冲突问题
Ubuntu 系统内自带了一个很久的 yarn ,在
`/usr/bin/yarn`
经过实验 无法在 ZetaChain 的 webapp 安装中使用

故查找系统环境变量下的所有 yarn
In: `which -a yarn`
Out:
```
/home/xboxpig/.nvm/versions/node/v20.19.6/bin/yarn
/usr/bin/yarn
/bin/yarn
```

并删除了`/bin/yarn`
In:
```
xboxpig@XBPG-PC:~/zetachin_proj/hello/frontend$ yarn
```
Out:
```

...

```
